Лабораторна робота №1

Секція 1: Теги тексту

Теги заголовків (h1-h6) структурують контент, p створює абзаци, а strong та em використовуються для логічного наголосу.

Заголовок 3-го рівня

Це абзац тексту з важливим фрагментом та курсивом.
А це новий рядок.

<h3>Заголовок 3-го рівня</h3>
<p>Це абзац з <strong>жирним</strong> та <em>курсивом</em>.<br>Новий рядок.</p>

Секція 2: Списки

Марковані (ul) та нумеровані (ol) списки використовуються для переліку елементів (li).

  • Елемент маркованого списку
  • Ще один елемент
  1. Перший пункт
  2. Другий пункт
<ul>
  <li>Маркований</li>
</ul>
<ol>
  <li>Нумерований</li>
</ol>

Секція 3: Посилання та зображення

Тег 'a' створює гіперпосилання, а 'img' вбудовує зображення. Атрибут alt важливий для доступності.

Пошук Google (target="_blank") 

<a href="https://google.com" target="_blank">Google</a>
<img src="..." alt="Опис зображення">

Секція 4: Таблиці

Таблиці структурують дані за допомогою рядків (tr) та комірок (td, th).

Ім'я Роль
Іван Студент 
<table>
  <tr><th>Ім'я</th><th>Роль</th></tr>
  <tr><td>Іван</td><td>Студент</td></tr>
</table>

Секція 5: Форми

Форми дозволяють користувачам вводити дані (input) та відправляти їх (button).

<form>
  <label for="email">Email:</label>
  <input type="email" id="email">
  <button>Надіслати</button>
</form>

Секція 6: Box Model

Кожен елемент — це прямокутник, який має відступи (margin), межі (border) та внутрішні відступи (padding).

Контент (Box Model)
.box {
  margin: 20px;   /* Зовнішній відступ */
  border: 5px solid yellow; /* Межа */
  padding: 15px;  /* Внутрішній відступ */
}

Секція 7: Typography

CSS дозволяє керувати розміром (font-size), жирністю (font-weight) та міжрядковим інтервалом (line-height).

Цей текст має змінену типографіку.

.text {
  font-size: 20px;
  font-weight: bold;
  line-height: 1.5;
}

Секція 8: Colors & Background

Властивості color та background-color задають кольори тексту та фону.

Білий текст на градієнтному фоні
.element {
  color: white;
  background: linear-gradient(to right, tomato, gold);
}

Секція 9: Layout (Flexbox)

Flexbox — це модель макету для розподілу простору між елементами в контейнері.

1
2
3
.container {
  display: flex;
  gap: 10px;
  justify-content: center;
}

Секція 10: Псевдокласи (:hover та :focus)

:hover спрацьовує при наведенні курсору, а :focus — коли елемент обрано (наприклад, клік по полю вводу). 



/* :hover для кнопки */
.btn:hover {
  background-color: white;
}

/* :focus для input */
.input:focus {
  border-color: #42f5aa;
  outline: none;
  background-color: #444;
}

Секція 11: Маніпуляція DOM

JavaScript дозволяє динамічно змінювати контент сторінки. Натисніть кнопку, щоб додати текст.

btn.onclick = function() {
  div.textContent = "Новий текст додано!";
}

Секція 12: Події (Зміна теми)

JS може змінювати класи CSS "на льоту". Кнопка нижче перемикає світлу тему для цієї секції.

Поточний стиль контейнера.

btn.onclick = function() {
  element.classList.toggle("light-theme");
}
